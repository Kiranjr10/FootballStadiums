### ğŸš€ How to Deploy to GitHub

To get your site live on `username.github.io/FootballStadiums/`:

#### 1. Push to GitHub
If you haven't yet, run these commands in your project folder:
```bash
git init
git add .
git commit -m "Finalized Footework - Nepal Edition"
```
Then, create a repository on GitHub named `FootballStadiums` and link it:
```bash
git remote add origin https://github.com/YOUR_USERNAME/FootballStadiums.git
git branch -M main
git push -u origin main
```

#### 2. Enable GitHub Pages
1. Go to your repository on GitHub.com.
2. Click **Settings** (top tab).
3. Click **Pages** (left sidebar).
4. Under **Build and deployment**, set the Branch to `main` and Folder to `/ (root)`.
5. Click **Save**.

#### 3. Access Your Site
Once deployed, your site will be available at:
`https://YOUR_USERNAME.github.io/FootballStadiums/fotework.html`

> [!IMPORTANT]
> Because the main file is named `fotework.html` (and not `index.html`), you must include `/fotework.html` at the end of your URL to see the homepage.
